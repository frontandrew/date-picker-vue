# Date Picker 0.0.0

### About:
    Vue component for pick date.

### Setup:

> it is preferable to use the [pnpm](https://pnpm.io/) as a dependency manager

- to correct setup dependencies use: `pnpm install --frozen-lockfile`
- to start dev server localy use: `pnpm dev`
- to build project use: `pnpm build`
- to serve builded project localy: `pnpm preview`
